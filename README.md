Stock App

This is my final year project for the BS Computer Science Degree.

Stock App is a web app developed in PHP framework Laravel 8.

It is basically a stock management system used to store the product details of the inventory of small to medium level businesses.

Stock app is full fledge app with backend and data base integration. PgSQL is used for storing the app data.

In this app user can create account an account, register the business and save the product details (quantity, price, dates etc)
